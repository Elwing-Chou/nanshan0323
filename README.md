### 安裝函式庫被防火牆擋下

請改用下指令的方式: 1. 在pycharm下的terminal tab (如果是venv建議使用)  2. 直接使用命令列

```
pip install --trusted-host=pypi.org --trusted-host=files.pythonhosted.org --trusted-host=pypi.python.org <package name>
```

``
python.exe -m pip install jieba
```
